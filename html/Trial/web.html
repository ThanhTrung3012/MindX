Làm quen giữa thầy và trò

⏱️ Thời gian: 5 phút

Mục tiêu: Xây dựng mối quan hệ thầy trò và tạo không khí học tập thoải mái.

Hoạt động:

Giới thiệu thầy và trò:

  • Thầy giới thiệu bản thân, công việc, và lý do chọn môn học này
  • Học viên cũng giới thiệu bản thân, bao gồm tên, trường học, sở thích cá nhân

Ví dụ câu hỏi:

  • "Em đã biết và tiếp xúc với những website nào?"
  • "Em nghĩ rằng để tạo ra 1 trang web có khó không, sử dụng nhiều kiến thức không?"

Câu hỏi khởi động về AI:

  • "Em hiểu gì về trí tuệ nhân tạo (AI)?"
  • "Em đã từng sử dụng AI trong học tập hoặc công việc chưa? Nếu có, hãy chia sẻ trải nghiệm đó."
  • "Em nghĩ AI có thể thay thế con người trong những lĩnh vực nào?"
  • "Theo em, điều gì khiến AI ngày càng được ứng dụng rộng rãi?"

Giới thiệu về phần mềm Visual Studio Code

⏱️ Thời gian: 10 phút

Mục tiêu: Giới thiệu Visual Studio Code và HTML, CSS, JS.

Giới thiệu tổng quan về Visual Studio Code:

Định nghĩa: Là một công cụ lập trình được hỗ trợ bởi nhiều ngôn ngữ lập trình, trong đó có HTML, CSS, JavaScript. VS Code có thể giúp tạo ra ứng dụng, website, dựa trên ngôn ngữ lập trình phù hợp.

Các tính năng chính:

  • Chạy trên nền tảng Windows và Mac
  • Gợi ý mã thông minh (IntelliSense): Tự động hoàn thành mã, hiển thị thông tin hàm
  • Gỡ lỗi (Debugging): Tích hợp công cụ gỡ lỗi ngay trong trình soạn thảo
  • Quản lý mã nguồn Git: Có thể commit, pull, push trực tiếp từ giao diện
  • Tiện ích mở rộng (Extensions): Có thể cài thêm công cụ hỗ trợ cho framework, ngôn ngữ hoặc style (ví dụ: Prettier, ESLint, C/C++ Extension...)

Thực hành cơ bản:

  • Mở VS Code, tạo các file liên quan
  • Giải thích ý nghĩa chức năng của từng file

Hiểu cấu trúc HTML và CSS cơ bản

⏱️ Thời gian: 25 phút

1. Phân tích cấu trúc trang HTML

Mục đích: Học viên nắm được cấu trúc một file HTML cơ bản và ý nghĩa của từng thành phần.

Chi tiết các bước thực hiện:

  • Mở file index.html
  • Giải thích các thẻ chính:
    • <!DOCTYPE html>, <html>, <head>, <body>
    • <nav>, <div id="intro">, <h1>, <a>, <img>, <p>, <ul>, <footer>...
  • Tìm hiểu cách liên kết tới CSS (<link rel="stylesheet">) và JS (<script>)

2. Xem và sửa CSS để thay đổi giao diện

Mục đích: Hiểu vai trò của CSS và biết cách thay đổi màu sắc, căn lề, font chữ...

Chi tiết các bước thực hiện:

  • Mở file style.css
  • Chỉnh thử:
    • Màu nền của phần <body>
    • Căn giữa đoạn tiêu đề <h1>
    • Bo tròn ảnh bằng thuộc tính border-radius
  • Gợi ý thuộc tính: color, background-color, text-align, font-size, margin, padding, width, height, font-weight,...

Thêm tương tác bằng JavaScript

⏱️ Thời gian: 10 phút

1. Hiểu cách chatbot hoạt động

Mục đích: Biết cách JavaScript thêm chức năng cho web.

Chi tiết các bước thực hiện:

  • Hướng dẫn thực hành gửi tin nhắn bằng nút:
    • Mở main.js
    • Giải thích các phần:
      • Gán sự kiện cho nút #chat-toggle-btn
      • Gửi và hiển thị tin nhắn trong #chat-box
  • Hướng dẫn thực hành gửi tin nhắn bằng nút enter:
    userInput.addEventListener("keypress", function(event) {
      if (event.key === "Enter") {
        sendBtn.click();
      }
    });
  • Có thể hướng dẫn các bạn tạo API key thay vì lập trình bên trên
